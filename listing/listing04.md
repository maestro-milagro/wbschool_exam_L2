Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Программа выведет все числа от 0 до 9 после чего будет deadlock. Произойдет это из за того что после начала работы программы основная горутина и горутина анонимной функции будут поочередно лочиться в ожидании на чтение или запись, и после того как анонимная фунуция выполнит свой код основная горутина залочиться навсегда, т. к. нет горутин которые моглы бы что-то записать в канал.

```
