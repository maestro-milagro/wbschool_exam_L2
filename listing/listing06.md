Что выведет программа? Объяснить вывод программы. Рассказать про внутреннее устройство слайсов и что происходит при передачи их в качестве аргументов функции.

```go
package main

import (
	"fmt"
)

func main() {
	var s = []string{"1", "2", "3"}
	modifySlice(s)
	fmt.Println(s)
}

func modifySlice(i []string) {
	i[0] = "3"
	i = append(i, "4")
	i[1] = "5"
	i = append(i, "6")
}
```

Ответ:
```
Выведет [3 2 3]. Т. к. слайс это структура хронящая указатель на массив, а также длину и ёмкость этого массива то даже в случае передачи значения слайса в функцию при обращении к элементам этого слайса мы все ещё будем обращаться к элементам основного массива. Поэтому мы без проблем можем изменить любое значение элеманта в фунуции и оно будет сохранено для изначального слайса. Проблемы возникают когда мы пытаемся увеличить количество элементов в слайсе т. к. для этого нам нужно присвоить новое значение изначальному слайсу. Из за того что в функции мы работаем с копией изначального массива, а аппенд при выходе за ёмкость в свою очередь создаёт новый массив при аппенде копии будет присвоено новое значение с ссылкой на другой массив и далее все изменения элементов будут происходить с новым массивом, а изначальный слайс останется нетронутым.

```
